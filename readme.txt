== Description ==
"Notification Bar Builder for Elementor" - Allow you to build your desired responsive notice in a quick time using the Elementor page builder. You can build the notifications on your own OR you may use the prebuilt templates. You can use the notification bar for notifying discounts, important updates, grab user's attention, present promotion, and any other content that might catch their interest.

You can see the live demo here : [Live Demo](https://demo.azplugins.com/notification-bar/)

Donwnload prebuilt templates from [here](https://demo.azplugins.com/notification-bar/documentation/notification-templates.zip) and then follow [This instruction](https://demo.azplugins.com/notification-bar/documentation/#config_2)

== Installation ==
This section describes how to install the "Notification Bar Builder for Elementor" plugin and get it working.

= 1) Install =

i. Go to the WordPress Dashboard "Add New Plugin" section.

ii. Search For "Image Hover Effects For Elementor".

iii. Install, then Activate it.

= OR: =

i. Unzip (if it is zipped) and Upload `notification-bar-builder-for-elementor` folder to the `/wp-content/plugins/` directory

ii. Activate the plugin through the 'Plugins' menu in WordPress

= 2) Configure =
i. After install and activate the plugin you will get a notice to install Elementor Plugin ( If allready have it then do not show any notice. ).

ii. To install the plugin click on the "Button" Install Elementor.

iii. Follow the screenshots to learn how it works!
